# Hostel Management System

A full-stack Hostel Management System developed using Spring Boot, React, and MySQL to manage hostel operations efficiently.

## Features

### Admin
- Manage Rooms
- Manage Students
- View Complaints
- View Leave Requests
- Dashboard Overview

### Warden
- Monitor Rooms and Students
- Resolve Complaints
- Approve/Reject Leave Requests
- Dashboard Statistics

### Student
- Submit Complaints
- Apply Leave Requests
- View Personal Dashboard

## Tech Stack

### Frontend
- React.js
- React Router
- CSS
- Axios

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Hibernate

### Database
- MySQL
